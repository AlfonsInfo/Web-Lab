pseudoclass adalah sebuah kelas yang semua yang dimiliki oleh sebuah eleme HTML, yang membuat ktia bisa mendefinisikan style tertentu pada "keadaan tertentu" oleh karena itu disebut pseudoclass

Contoh pseudoclass :

  1. link
  2. hover
  3. active
  4. visited

pseudo-class berhubungan dengan posisi elemen

  1. first-child
  2. last-child
  3. nth-child(n)
Contoh Link
