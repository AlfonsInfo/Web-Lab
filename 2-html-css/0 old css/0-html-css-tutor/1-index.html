Apa itu css ?

CSS adalah singkatan dari Cascading Style Sheet. Menurut W3SCHOOL adlaah mekanisme sederhana yang mengatur gaya.

Bagaimana cara melihat teknologi yang digunakan ? coba matiakn dependensinya

Cascading Style Sheet !!

  1. Aturan yang digunakan untuk menampilkan elemen/ tag html
  2. Dibuat terpisah dengan HTML
  3. bertujuan untuk memisahkan konten dan style
  4. 1 CSS dapat digunakan untuk banyak halaman HTML
  5. 1 halaman HTML dapat terlihat berbeda jika menggunakan CSS yang berbeda pula

Anatomi CSS

  1. selector property : value
  2. selector untuk memilih dan memanipulasi elemen spesifik pada HTML
  3. elemen HTML dipilih berdasarkan tag, id , class bahkan pola/ pattern
  4. semakin kompleks struktur HTML, selector juga bisa semakin kompleks/ spesifik
  5. Property & Value 350 ++
  6. Referensi : https-tricks.com/almanac

Penempatan CSS

  1. embed menggunakan tag style pada head
  2. inline menggunakana attribut style
  3. external : membuat file css secara terpisah
