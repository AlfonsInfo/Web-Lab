Asynchronous Function

Sebuah function ynag bekerja secara asynchronous melalui event loop yang menghasilkan implisit promise sebagai return value, tapi cara penulisan code menggunakan penulisan yang standard

  1. Fungsi berjalan secara asynch
  2. Menghasilkan implisit promise
  3. Cara penulisan sama dengan synchronous function

Sebuah async function dapat meiliki keyword await di dalamnya untuk memberhentikan sementara eksekusi fungsinya sambil menuggu promisenya selesai
