Function iki opo ?

  1. sebuah sub-program yang dapat dipanggil pada bagian laind ari program
  2. merupakan struktur dasar pembentuk javascript
  3. termasuk ke dalam first class object

Kategori Function ?

  1. Built-in Function
  2. User-Defined Function

User defined function

  1. Fungsi yang dibuat sendiri
  2. Menggunakan keyword function
  3. Nama Fucntion (Optional)
  4. Parameter / Argument
  5. Body dibungkus dengan {}
  6. Bisa mengembalikan nilai atau tidak
