Penjelasan mengenai table

Ada beberapa table tag :

  1. table defines a table
  2. th stand for table head
  3. tr stand for table row
  4. td stand for table data
  5. caption defines a table caption
  6. colgroup specifies a group of one or more columns in a table for formatting
  7. col defines specifies column properties for each column within a colgroup element
  8. thead group the header content in a table
