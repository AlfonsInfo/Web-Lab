Combining Selector

Konsep :

Ada beberapa metode mengkombinasikan selector

  1. Group (,) Apply to both selector (aplikasikan style terhadap >= 2 elemen yang di seleksi)
  2. Child (>) Apply style to child (one gen) terapkan style ke anakan elemen
  3. Descendant( (spasi)) (Apply pada turunan)
  4. Chaining(selectorselector) (Apply ketika semua syarat seleksi true)
